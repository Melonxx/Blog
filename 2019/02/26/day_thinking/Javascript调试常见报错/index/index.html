Javascript 调试常见报错

2019-02-26 | 日常杂疑
Contents

ECMA-262 定义了下列 7 种错误类型：

  1. Error 错误

  2. EvalError 全局错误

  3. ReferenceError（引用错误） 对象表明一个不存在的变量被引用。

  4. RangeError 对象标明一个错误，当一个值不在其所允许的范围或者集合中。

  5. SyntaxError（语法错误）对象代表尝试解析语法上不合法的代码的错误。

  6. typeError（类型错误）对象用来表示值的类型非预期类型时发生的错误

  7. URIError 编码错误
