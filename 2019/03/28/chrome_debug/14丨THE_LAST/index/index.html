    高度推荐你订阅 Google Chrome Developers 的 YouTube 频道，这是在 Chrome 中展示已有的新内容的地方。 不仅如此，你还可以去那里寻找有关最佳实践，JavaScript 演变等的讨论。对于任何一位前端开发人员来说，这里都有许多有价值的内容。
