五十、 :read-only

表示元素不可被用户编辑的状态（如锁定的文本输入框）

与 :read-write 相反

五十一、 :read-write

注意：这个选择器不仅仅选择 <input> 元素，它也会选择所有可以被用户编辑的元素，例如设置了 contenteditable 属性的  <p> 元素。
