    #48个音

    /i/ 上下牙之间小拇指

    sit /sit/
    busy /bizi/ （当口语时会以a结尾）

    /l/ 舌尖顶上齿龈

    world /we:ld/
    word /we:d/

    /il/ !== /u:/

    build /bild/
    milk /milk/

    /r/ 圆唇，卷舌，吐气

    rose 不读 肉丝 读 若哦(延长)子

    口语技巧

    重叠现象 / 辅音➕元音

    Class starts in 20 minutes

    重叠

    roommate
    what time
    take care

    辅音➕元音(加粗部分连读(是连读，不是省略！))

    take off
    stand up

    重叠现象 / 失去爆破

    I don’t want to do like this (口语want 和 to 在一起时读wanan)

    失去爆破 （p b t d k g）(连续出现时，前者省略)

    blackborad
    goodbye
    stop talking
    good time

    同化现象 / 失去爆破 / 重叠现象 / 辅音➕元音

    Would you like to talk about this friend （口语friend d轻读或不读）

    认知连读方式

    1.找–辅音➕元音
    2.找–失去爆破
    3.找–重叠，同化
    4.找–美式浊化（美式英语）🌰：当t在两个元音之间时读 d 如： water => wader
