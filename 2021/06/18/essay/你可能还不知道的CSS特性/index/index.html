    10. CSS 比较函数 min() 和 max() 和 clamp()

    🌰：min(50vw, 500px) 在浏览器视窗宽度改变时，返回的值的变化：

    clamp(MIN, VAL, MAX)，这三个值之间的关系（或者说取值的方式）：
    • 如果 VAL 在 MIN 和 MAX 之间，则使用 VAL 作为函数的返回值
    • 如果 VAL 大于 MAX ，则使用 MAX 作为函数的返回值
    • 如果 VAL 小于 MIN ，则使用 MIN 作为函数的返回值
